# Metaobject limits

This document lists how many metaobject definitions and entries can be created per shop, including apps, by plan.

### Metaobject definition limits

Note

[Standard metaobject](https://shopify.dev/docs/api/admin-graphql/latest/objects/StandardMetaobjectDefinitionTemplate) definitions don't count towards limits.

#### App definitions

Each app installed on a shop can create up to 128 metaobject definitions.

#### Merchant definitions

The merchant's Shopify plan limits how many metafield object definitions they can create.

| Plan | Maximum allowed definitions |
| - | - |
| Basic | 128 |
| Shopify | 128 |
| Advanced | 128 |
| Plus | 256 |
| Enterprise | 256 |

Note

Definitions configured in TOML configuration are app scoped.

### Metaobject entry limits

Note

[Standard metaobject](https://shopify.dev/docs/api/admin-graphql/latest/objects/StandardMetaobjectDefinitionTemplate) entries don't count towards limits.

Each metaobject definition can have up to 1,000,000 entries.
